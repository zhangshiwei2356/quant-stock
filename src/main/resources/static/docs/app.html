Quant Stock 是基于 Spring Boot 2.7 + TA4J 的 A 股量化回测与模拟交易单体应用（非前后端分离）。页面在 static/，接口在 /api/**，数据默认落本地 MySQL quant_stock。

如何启动

  • 项目根目录：执行 mapper/schema.sql 后 mvn spring-boot:run
  • 浏览器：http://localhost:8080/stock.html
  • 空库自动导入日线 + 5 分钟种子；默认 market-mode=db、trade-mode=sim

架构一览

  • 前端：stock.html + stock-chart.js；文档片段 static/docs/*.html
  • 后端包：market 行情 · strategy 策略 · backtest 回测 · pool 目标池 · trade 下单账本 · risk 风控 · task 调度 · account/admin 账户与运维 · pdf 文档
  • 数据：行情表 + 目标池 + 模拟账本 + 回测历史 + sys_schedule_job
  • 完整架构图与接口表见侧栏「项目 README」（实时渲染仓库 README.md）

功能模块

  • 行情浏览：全市场模糊选股、多周期 K 线与指标
  • 个股回测：回测工作台 / 批量扫描 / 回测历史
  • 组合回测：回测工作台 / 回测历史（共享资金池日 K）
  • 目标池：当前池 / 扫描历史；盘后或手动扫描覆盖唯一池；移出≠卖出
  • 账户概览：资金 · 持仓（T+1 批次）· 委托 · 日结 · 风控事件
  • 运维中心：任务管理 / 数据健康 / 运行参数（任务种子默认全关）
  • 数据表：白名单表分页只读
  • 量化知识 / 应用说明：指标与规则文档；应用说明含系统概述、项目 README、交易规则、待办、宽睿梳理；可导出 PDF

模拟数据与默认参数

  • 演示股：600036 / 000001 / 300059；约 2025-07-17 ~ 2026-07-17
  • 物理表：market_daily、market_minute；回测：bt_backtest_*
  • 回测区间默认留空 = 全量 K 线；初始资金默认 100000
  • 页头主题默认日间（夜盘 / 银河 / 极光可切换并记住）

策略与风控摘要

  • 均线金叉死叉；可配 MA60 / 放量 / ADX / RSI
  • 综合成本止损、移动止盈；T+1 分档；金字塔 50/30/20
  • 撮合：日 K 下一根开盘；分钟 ≥ 次日 09:45；涨跌停主板 10% / 创科 20%
  • 单股 / 组合 / 扫描交易对齐核心规则；实盘为本地模拟账本（真柜台见待办 / 宽睿文档）
  • 细则：「应用说明 → 交易规则」

可选能力

  • 定时总闸 quant.schedule.enabled；API：GET /api/schedule
  • 安全：可选 QUANT_API_KEY；限流 QUANT_RATE_LIMIT
  • 扩展：KlineSdkClient / trade-mode=sdk（当前为桩）

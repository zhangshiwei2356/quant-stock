Quant Stock 是基于 Spring Boot 2.7 + TA4J 的 A 股量化回测单体应用（非前后端分离），页面内嵌于 static/。

如何启动

  • 项目根目录执行：mvn spring-boot:run
  • 浏览器打开：http://localhost:8080/stock.html
  • 默认连接本地 MySQL（localhost:3306/quant_stock，root/123456），并执行 mapper/schema.sql
  • 空库启动时自动从 classpath JSON 导入日线 + 5 分钟模拟数据

行情与回测存储

  • 股票：600036 招商银行、000001 平安银行、300059 东方财富
  • 区间：约 2025-07-17 ~ 2026-07-17
  • 物理表：market_daily（日线）、market_minute（5分钟）；其它周期运行时聚合
  • 回测历史/分析：bt_backtest_record / bt_backtest_analysis
  • 种子目录：src/main/resources/data/kline/（仅导入用）

页面能做什么

  • 进入应用先显示初始化页；展开左侧一级菜单进入功能，再点同一菜单可全部收起并回到初始化页
  • 行情浏览：仅行情展示，可同时开启多只股票信息（标签切换）
  • 个股回测：选股后可见基础K线+信号图（标注买卖点）+权益曲线+成交明细/收益汇总+K线表；历史与分析一一对应，点击历史行在该行下方展开决策分析
  • 组合回测：勾选多只共享资金池回测；权益曲线 + 收益看板 + 成交流水 + 分股汇总；点击历史行在该行下方展开对应分析
  • 回测起止时间默认留空 = 全量 K 线（单股/组合均支持填写）；初始资金默认 100000
  • 侧栏另有量化知识（指标/涨跌停/T+1/成本/仓位/风控/撮合等）与应用说明（介绍、交易规则、数据待办）；正文在 static/docs/，载入本面板
  • 页头主题：黑客帝国 / 科技粒子 / 青松·3D星空 / 星空粒子，本地记住选择

策略与风控摘要

  • 均线金叉死叉；可配置 MA60 / 放量 / ADX / RSI 过滤
  • 综合成本止损、移动止盈；T+1 分档；金字塔 50/30/20（成交后占档）
  • 撮合双口径：日K下一根开盘；分钟K≥次日09:45；涨跌停主板10%/创科20%
  • 单股 / 组合 / 实盘扫描均对齐上述核心规则（实盘为模拟现金账本）
  • 细则见「应用说明 → 交易规则」

可选能力

  • 安全：可选 QUANT_API_KEY；限流 QUANT_RATE_LIMIT
  • 市值股本：quant.float-shares-yi.<code>
  • 扩展：KlineSdkClient / trade-mode=sdk

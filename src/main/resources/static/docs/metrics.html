评价策略时，除了「赚了多少」，还要看回撤、胜率与交易频率，避免只看终点收益。

本应用统计口径

  • 权益 = 现金 + 持仓市值（已含交易成本）
  • 总收益率 = (期末权益 − 初始资金) / 初始资金
  • 峰值回撤 = (历史最高权益 − 当前权益) / 历史最高权益
  • 胜率 = 盈利完整开平回合数 / 全部完整开平回合数（非按每笔成交）
  • 综合成本（单票）= Σ(买入成交额 + 买入佣金) / 股数
  • tradeStats：买卖次数/股数/手数/成交额、费用合计、总盈亏等

阅读建议

  • 高收益伴随高回撤时，关注熔断是否触发、仓位是否过大
  • 胜率高但盈亏比差，或交易次数极少，都要结合权益曲线看
  • 组合回测另有分股贡献；批量扫描按总收益率排序，且与有效回测门槛不完全相同

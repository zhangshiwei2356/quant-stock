A股多数股票设有涨跌停限制：当日价格相对昨收不得超过规定幅度。触及后流动性变差，买单难买、卖单难卖。

常见幅度

  • 主板 / 中小板：约 ±10%
  • 创业板、科创板：约 ±20%
  • ST 等另有规则；本演示池未单独建模 ST 幅度

本应用相关

  • 相对上一交易日收盘判定（分钟序列不取相邻 bar 的昨收）
  • 软阈值：收盘触及昨收×(1±限幅×0.95) 即判涨/跌停
  • 封板：开高低收均贴齐限价 ±0.01
  • 涨停：禁止开仓/加仓成交；跌停：挂卖可能失败，连续 3 个交易日失败后按跌停价×0.99 强平
  • 停牌（量≤0）禁止开仓；复牌无「首日必卖」专项规则

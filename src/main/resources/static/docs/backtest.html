回测是用历史 K 线模拟策略买卖，评估收益、回撤与胜率等。结果不等于实盘收益。

常见陷阱

  • 未来函数：用到当时未知的信息（如未收盘价、未来均线）
  • 成本低估：忽略佣金、印花税、滑点、冲击，会高估收益
  • 过拟合：参数只拟合一段历史，换区间失效

本应用相关

  • 信号收盘确认后，默认按下一根开盘（或分钟次日≥09:45）撮合，降低未来函数
  • 成本含佣金、卖出印花税、分级滑点与冲击（见「交易成本」）
  • 支持单股 / 组合 / 批量扫描；初始资金默认 100000；时间留空=全量 K 线
  • 单股引擎约需 ≥65 根 K；批量扫描列表门槛为 ≥20 根（中间区间可能空结果）
  • 历史与决策分析写入 MySQL；组合/实盘与单股有差异，见「交易规则」

ATR（Average True Range，平均真实波幅）衡量价格波动幅度，不区分涨跌方向，常用周期 14。

  • ATR 大：波动剧烈，止损需更宽，仓位宜更小
  • ATR 小：波动温和，止损可收紧；过小则流动性/波动不足，策略可能拒绝开仓

本应用相关

  • 开仓过滤：ATR14 须 > atr-min-threshold（默认 0.001）
  • 动态仓位：ATR调节 = baseAtr(0.05) / 当前ATR，夹到 [0.2, 1.5]，波动大则买得更少
  • 止损：成本 − 2×ATR，并与「权益×2%/股数」硬止损取更严（更高）的止损线，只上移
  • 移动止盈：持仓最高价 − 1.5×ATR
  • 单股引擎 ATR 缺失时回退 baseAtr；组合/实盘缺失多为 0（通常无法开仓）

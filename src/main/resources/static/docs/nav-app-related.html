应用说明收录启动方式、交易规则、待办清单与宽睿对接梳理，便于对照代码能力。

  • 本菜单包含
    • 系统概述：启动、模块边界、库表与页面结构
    • 交易规则：回测/扫描对齐的撮合、成本、仓位与风控
    • 待办清单：本地已落地项 + 依赖外部 API 的待办（整应用路线图）
    • 宽睿文档梳理：本地 Quant360 OES/MDS 资料包能力与本应用缺口对照
  • 怎么用
    • 在左侧点选「系统概述 / 交易规则 / 待办清单 / 宽睿文档梳理」
    • 改代码后优先核对「待办清单」状态是否仍准确

可将系统概述、交易规则、待办清单、宽睿文档梳理合并导出为 PDF（服务端生成）。再点一次「应用说明」可收起并回初始化页。

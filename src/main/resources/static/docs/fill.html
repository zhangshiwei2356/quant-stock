撮合规则决定「信号出现后以什么价格、何时成交」。不合理的撮合会引入未来函数或过于乐观的成交。

常见口径

  • 信号当根收盘成交（简单但易偏乐观）
  • 信号下一根开盘成交（更常见、更保守）
  • 限价挂单、触及成交等（更接近实盘，建模更复杂）

本系统用法

  • 默认 next-bar-open-fill=true：日K/周月 → 下一根开盘；分钟 → 信号日次日且 ≥09:45
  • 设为 false 时按当根收盘撮合
  • 止损 / trail：当根触及，基准价 max(止损价, 最低价) 再套卖滑点
  • 静默时段（分钟）禁止挂新开；买单过滤未过时挂单可保留重试，+5 日历日过期
  • 撮合步内先卖后买；详见「交易规则」第五、七节

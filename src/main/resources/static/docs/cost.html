实盘与回测都应计入交易成本，否则收益会被高估。常见项：佣金、印花税、买卖价差/滑点、大单冲击。

概念

  • 佣金：券商按成交额收取（常见万三量级，实盘另有最低 5 元等，本应用未建模最低佣金）
  • 印花税：目前通常仅卖出收取
  • 滑点：理想价与实际成交价的偏差
  • 冲击成本：大单推动价格对自己不利的额外成本

本系统用法

  • 佣金默认万三（fee-rate）；印花税默认千一仅卖（stamp-tax-rate）
  • 分级滑点（近 20 均量）：≥2000万→0.05%；≥500万→0.2%；否则→0.5%
  • 冲击：min(0.1×本笔量/20日均量, 2%)；买上浮、卖下浮，与滑点叠加
  • quant.slip-point 不参与定价，真实滑点以分级为准
  • 权益与盈亏统计均在扣除上述成本之后

移动平均线（Moving Average）是一段时间收盘价的算术平均，用来刻画趋势与成本重心。

  • MA5：近 5 根均价，更灵敏，短线成本参考
  • MA20：近 20 根，中短线趋势参考
  • MA60：近 60 根，偏大周期方向过滤

金叉 / 死叉

金叉：短期均线由下向上穿越长期均线；死叉：短期由上向下穿越长期。

  • 本策略：上一根 MA5≤MA20，本根 MA5>MA20 → 金叉（收盘确认）
  • 死叉：上一根未处于 MA5<MA20，本根 MA5<MA20

本系统用法

  • 策略名 MA_CROSS_FILTERED：金叉买入、死叉卖出全部持仓
  • 金叉后还须通过 RSI/ATR 等必过过滤；可选开启 MA60 上行过滤（只看均线抬升，不要求价在 MA60 上方）
  • 金字塔加仓要求当时仍保持 MA5>MA20
  • 震荡市假交叉多，可叠加放量、ADX 过滤（默认关闭，见配置）

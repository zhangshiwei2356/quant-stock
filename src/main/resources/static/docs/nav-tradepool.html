管理量化交易的唯一目标池。二级菜单查看当前池名单与历次扫描入选报告；盘后扫描自动覆盖；人工可移出（移出≠卖出）。

二级菜单说明

  • 当前池：当前生效的可买名单（综合分、入池原因）
  • 扫描历史：按批次查看历次入选；也可在此手动「扫描更新」
  • 扫描更新：全市场扫描并覆盖当前池，写入新批次；可下载 Markdown 报告
  • 刷新列表：只重载页面数据，不发起扫描

步骤建议流程

  1. 在「当前池」或「扫描历史」点「扫描更新」，或开启定时 pool-rebuild
  2. 不需要的标的在当前池点「移出」（移出≠卖出）
  3. 在「扫描历史」核对某批次入选名单

再点一次「目标池」可收起并回初始化页。

二、依赖外部 API / 数据源（暂不能补）

  1. 真实行情 API（优先对接宽睿 MDS） — 未接入 · market-collect 仍为骨架。待接：MdsClient.subscribeMarketData、L1/L2 快照回调、qryMktDataSnapshot / qrySnapshotList；自聚合成分钟/日线后落库
  2. 券商交易 API（优先对接宽睿 OES） — 未接入 · quant.trade-mode=sdk 仅为状态桩。待接：sendOrdReq / sendBatchOrdsReq / sendOrdCancelReq + OesCallBack 委托/成交回报与 initRptSync
  3. 券商委托/持仓对账（OES 查询） — 未接入 · sync-orders ← queryOrder/queryTrade；position-pnl-sync ← queryCashAsset/queryStkHolding（及可选 queryClientOverview）
  4. 证券状态 / 静态信息（MDS+OES） — 部分本地粗滤 · 待接 MDS qrySecurityStatus、qryStockStaticInfo(List)（涨跌停价、停牌标识）；OES queryStock
  5. 交易日 / 市场时段（OES+MDS） — 部分静态表 · 待接 OES queryTradingDay / queryMarketState；MDS qryTrdSessionStatus
  6. 真实佣金费率（OES） — 未落地 · 待接 queryCommissionRate（替换配置近似费率）
  7. 银证转账（OES，真仓运维） — 未落地 · sendCashTrsfReq + queryCashTransferSerial（模拟账本可不接）
  8. 历史 K 线全市场源 — 部分：本地库/种子 · MDS 偏实时，不替代多年回测历史；需另源或自建落库管道
  9. 复权因子（除权除息）— 未落地 · 宽睿包未覆盖；需复权因子或前复权行情源
  10. 资金流向（主力净流入/北向）— 未落地 · 无专用接口；L2 逐笔可自算近似，北向需另源
  11. 真实市值/流通股本/日均额 — 部分启发式 · MDS/OES 静态信息可补一部分；完整股本与日均额或需基本面源
  12. 财报与估值（PE/ROE 等）— 未落地 · 宽睿不覆盖；需财务数据 API
  13. 新闻舆情 / 另类数据 — 未落地 · 宽睿不覆盖
  14. 两融 / 期权 / ETF 申赎 / 新股中签 — 未规划 · OES 有完整接口族，当前现货模拟策略范围外

一句总结：本地模拟闭环已可跑；对接宽睿时优先 MDS 行情 → OES 下单回报 → 委托/持仓对账；复权/财报/舆情仍要其他数据源。

除不同周期 K 线价量外，完整量化系统还需基本面、事件驱动、另类数据、行情衍生四大类。本页记录与金叉策略相关的数据待办；应用实质性改动后须对照更新（见项目规则）。

落地优先级（投入产出比）

  1. 复权因子（除权除息）— 未落地 · MA/ATR 不复权会失真
  2. 资金流向（主力净流入）— 未落地 · 金叉+流入增强置信度
  3. 市值/流动性（流通市值、日均成交额）— 部分：规则有、真数不足（现为股本启发式）
  4. 财报（PE、ROE、营收增速）— 未落地 · 建议金叉后再滤 PE/ROE
  5. 停牌/ST 状态— 部分：停牌量≤0；缺实时 ST/退市源
  6. 新闻舆情— 未落地 · 噪声大、需 NLP

一句总结：骨架已有；下一步优先复权因子与资金流向。

一、基本面数据（与金叉关联最密）

决定股票质地，可作买入前二次过滤（在 RSI<60、ATR>0.001 之上）。

子类数据项策略用法
估值PE / PB / PS / PEG过滤估值过高，避免追高
盈利ROE、毛利率、净利率避开业绩暴雷股
成长营收/净利同比增速成长股与金叉共振
财务健康资产负债率、现金流、应收避开高负债、现金流紧张
市值总市值、流通市值已有「市值过低→不开」，需真实阈值

实操建议：金叉后加滤 PE < 行业PE中位数×1.5 且 ROE > 10%。

ADX（Average Directional Index）衡量趋势强度，不直接告诉涨跌方向。常与 +DI/−DI 一起使用；本应用主要用 ADX 数值过滤震荡市。

  • 经验上 ADX 偏低多为震荡/无趋势，均线交叉假信号多
  • ADX 升高表示趋势相对明确，更适合趋势跟踪

本应用相关

  • 配置开关 adx-filter-enabled（yml 默认 关）
  • 开启后须同时满足：ADX ≥ adx-min（默认 25），且 ADX ≥ adx-chop-max（默认 20，低于视为震荡拒单）
  • 仅过滤金叉买入；死叉卖出不受 ADX 限制
  • 计算预热较长，故有效回测 K 线需约 ≥65 根
